Flowgorithm C语言fpgt模板
作者：samir
知识星球号：C语言C++汇编研究中心
免费加入地址：https://t.zsxq.com/jeYVJuj


官方网址：flowgorithm.org

优点：
1.动态交互式流程图式编程，支持25种语言，3种伪代码
2.可以设计函数
3.可以设置快中慢执行，能很好的用于教学
缺点：
1.数据类型有限，只有整型，实数型，字符串型，布尔型四种
2.无法支持指针类型
3.文档只有英文和意大利语言，好在程序支持中文，不影响使用

补充：
根据官方的文档，在Java的基础上修改成C语言模板，经过调试测试基本代码生成功能正常，
由于没有经过严格系统测试，可能存在未知bug,如果发现，欢迎指出。
